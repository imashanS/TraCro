TraCro — Define Your Peak

TraCro is a full-stack fitness tracking and analytics platform built with modern web technologies.
The project combines a premium SaaS-style frontend experience with a secure Spring Boot backend API for workout tracking, analytics, authentication, and performance monitoring.

⸻

🚀 Tech Stack

Frontend

* Next.js 14 (App Router)

* TypeScript
* Tailwind CSS
* Axios

Backend

* Spring Boot
* Java 21

* Spring Security
* JWT Authentication
* BCrypt Password Hashing

* Maven
